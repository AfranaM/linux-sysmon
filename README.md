# Linux SysMon

A simple Bash-based system monitoring utility built while learning Linux shell scripting.

## Features

* View system information
* View current user and logged-in users
* View disk usage
* View network information
* Interactive menu-driven interface
* Input validation for invalid menu options

## Commands Used

* `hostname`
* `uptime`
* `whoami`
* `who`
* `df -h`
* `ip a`

## How to Run

Make the script executable:

```bash
chmod +x sysmon.sh
```

Run the script:

```bash
./sysmon.sh
```
